

#  **Задание 1 — Односвязный список**

##  Цель  
Вставить значение **M** перед каждым вторым элементом односвязного списка.  
Вывести указатель на последний элемент.

##  Структура данных  
Реализован класс `LinkedList`, содержащий:

- `add_node(int)` — добавление в конец  
- `insert_before_every_second(int)` — вставка M  
- `print_list()` — вывод  
- `get_last_node()` — получение последнего узла  
- `clear_list()` — освобождение памяти  

Узел списка представлен структурой `Node`.

##  Алгоритм вставки  
1. Идём по списку.  
2. Счётчик `count` увеличивается на каждом шаге.  
3. Если `count % 2 == 0`, вставляем новый узел перед текущим вторым элементом.  
4. Продолжаем обход.

##  Защита от дурака  
- Проверка корректности ввода количества элементов.  
- Проверка корректности ввода каждого значения.  
- Проверка, что список содержит минимум 2 элемента перед вставкой.  
- Проверка на пустой список при выводе.  
- Безопасная функция `safe_input()` предотвращает краш при вводе текста вместо числа.

##  Соответствие требованиям  
- Односвязный список реализован вручную  
- Память освобождается  
- Вставка корректна  
- Указатель на последний элемент выводится  
- Дружественный интерфейс  
- Ввод с клавиатуры  
- FIX_ME‑комментарии встроены  
- Стиль Google соблюдён

#  **Задание 2 — Очередь (TQueue)**

##  Цель  
Добавить N элементов в очередь, затем извлечь 5 первых элементов с помощью `dequeue`.

##  Структура данных  
Реализована структура:

- `Node` — элемент очереди  
- `TQueue` — содержит `head` и `tail`  

Методы:

- `enqueue(TQueue&, int)`  
- `dequeue(TQueue&)`  

##  Алгоритм работы  
### enqueue:
- Если очередь пустая → head = tail = новый узел  
- Иначе tail->next = новый узел  

### dequeue:
- Удаляется head  
- head = head->next  
- Если очередь стала пустой → tail = nullptr  

##  Защита от дурака  
- Проверка корректности ввода количества элементов.  
- Проверка корректности ввода каждого значения.  
- Проверка на пустую очередь перед удалением.  
- tail корректно обнуляется.  
- Безопасная функция `safe_input()`.

##  Соответствие требованиям  
- Очередь реализована вручную  
- Память освобождается  
- Извлекаются 5 элементов  
- Адреса head и tail выводятся  
- Дружественный интерфейс  
- Ввод с клавиатуры  
- FIX_ME‑комментарии встроены  
- Стиль Google соблюдён  

##  Основные исправления (FIX_ME)

# **Задание 5 — Двусвязный список**

Узел — структура `Node`.
